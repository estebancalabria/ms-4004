## English

### 📋 About This Course

**MS-4004** is a Microsoft official course designed for **executive-level professionals** who want to leverage the full potential of **Microsoft 365 Copilot** to drive strategic decisions, streamline communication, and automate routine work across the organization.

This course is part of the **Microsoft 365 Copilot** learning path and is intended for business leaders who want to use AI to enhance their productivity and organizational impact — without needing a technical background.

---

### 🎯 Target Audience

- C-Suite executives and senior leaders
- Department heads and project sponsors
- Anyone in a strategic leadership role using Microsoft 365

---

### 🏆 Course Objectives

By the end of this course, participants will be able to:

- Use **Copilot Chat in Microsoft Teams** to synthesize emails, meetings, and chats into actionable summaries
- Leverage **Copilot in Microsoft Word** to create and refine executive-level documents
- Analyze business data using **Copilot in Microsoft Excel**
- Build compelling presentations with **Copilot in Microsoft PowerPoint**
- Manage and track projects using the **AI Project Manager agent in Microsoft Planner**
- Create custom **AI agents** using Copilot Studio lite to monitor business KPIs proactively

---

### 🔑 Key Skills Covered

- Prompting techniques for executive workflows
- Synthesizing communications across Microsoft 365 apps
- Creating and refining business documents with AI assistance
- Data analysis and forecasting with Copilot
- Building and configuring custom AI agents
- Proactive business monitoring using Copilot agents

---

### 🛠️ Prerequisites & Requirements

- Microsoft 365 **Business Standard, Business Premium, E3, or E5** license
- **Microsoft 365 Copilot** add-on license (required for all exercises)
- Active OneDrive account
- Basic familiarity with Microsoft 365 apps (Word, Excel, Teams, Outlook)

> ⚠️ **Important:** This course **cannot** be completed with a Microsoft 365 Personal or Family subscription. A business/enterprise license with the Copilot add-on is required.

## Español

### 📋 Acerca del Curso

**MS-4004** es un curso oficial de Microsoft diseñado para **profesionales de nivel ejecutivo** que desean aprovechar todo el potencial de **Microsoft 365 Copilot** para tomar decisiones estratégicas, optimizar la comunicación y automatizar tareas rutinarias en toda la organización.

Este curso forma parte de la ruta de aprendizaje de **Microsoft 365 Copilot** y está dirigido a líderes empresariales que quieren usar la IA para potenciar su productividad e impacto organizacional, sin necesidad de conocimientos técnicos avanzados.
